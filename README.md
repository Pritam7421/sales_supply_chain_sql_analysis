# 📊 Sales & Supply Chain SQL Analysis

A business-focused SQL analytics project built on a **large-scale sales and supply chain dataset (~7M+ records)**.  
This project demonstrates how SQL can be used not just to query data, but to generate **real business insights** across **sales, finance, customer behavior, product performance, and supply chain forecasting**.

---

## 🚀 Project Overview

This project was developed to simulate how a Data Analyst works with enterprise-level business data to answer practical questions such as:

- Which customers and markets are driving the most revenue?
- Which products are performing best across divisions?
- How are sales trending over time?
- What does the Profit & Loss picture look like?
- How accurate are demand forecasts in the supply chain?

The project follows a **business-first analytics approach**, where raw transactional data is transformed into decision-ready insights using SQL.

---

## 🎯 Business Objective

The goal of this project is to analyze sales and supply chain operations in order to uncover:

- **Revenue trends**
- **Profitability patterns**
- **Customer and market contribution**
- **Top-performing products**
- **Forecasting accuracy**
- **Business growth opportunities**

---

## 🧩 Dataset Structure

The project is based on a **star schema model**, commonly used in real-world analytics and BI systems.

### Key Tables Used:
- `fact_sales_monthly`
- `fact_gross_price`
- `fact_forecast_monthly`
- `dim_customer`
- `dim_product`
- `dim_date`

This structure helped in building scalable analytical queries across multiple business domains.

> ⚠️ **Note:** The original dataset is not included in this repository due to sharing restrictions.

## 📌 Analysis Areas Covered

This project is divided into multiple business analysis modules:

### 1. Data Overview
Understanding the dataset structure, available tables, and key business entities.

### 2. Sales Analysis
Analyzing sales trends across months, years, products, and customers.

### 3. Financial Analysis
Evaluating revenue, gross sales, discounts, and profitability.

### 4. Customer & Market Analysis
Identifying high-value customers and top-performing markets.

### 5. Product Analysis
Finding best-selling products, division-wise performance, and category-level insights.

### 6. Supply Chain Analysis
Measuring forecast accuracy and identifying planning inefficiencies.

### 7. Contribution Analysis
Understanding how different segments contribute to overall business performance.
